**Project Overview**
This project predicts supply chain delivery delays using machine learning algorithms and Graph Neural Networks (GNN). The system analyzes logistics data such as vendor information, shipment mode, freight cost, and delivery timelines to identify delayed shipments and improve supply chain performance.
The project combines:
Machine Learning models
Graph Neural Networks
Explainable AI (SHAP)
Feature Engineering
SMOTE for class imbalance


**Objectives**
Predict delayed deliveries in supply chain
Identify factors affecting shipment delays
Compare machine learning models
Build supply chain network graph
Apply Graph Neural Network (GNN)
Provide explainable AI insights


**Technologies Used**
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
XGBoost
NetworkX
PyTorch
PyTorch Geometric
SHAP



**Machine Learning Models**
The following models were implemented:
Logistic Regression
Decision Tree
Random Forest
XGBoost
Graph Neural Network (GCN)



**Visualizations**
The project includes:
Late Delivery Distribution
Freight Cost Histogram
Correlation Heatmap
Confusion Matrix
ROC Curve
Feature Importance Plot
SHAP Summary Plot
Supply Chain Network Graph



**Model Performance**
Logistic Regression	65.6%
Decision Tree	91.6%
Random Forest	92.7%
XGBoost	92.8%

**Best Model: XGBoost**
